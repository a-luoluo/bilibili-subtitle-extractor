# B站字幕提取器 (Bilibili Subtitle Extractor)

一键提取哔哩哔哩视频字幕的 Chrome 浏览器扩展，支持阅读模式和时间戳跳转。

## ✨ 功能

- 🔍 一键提取 B 站视频字幕
- 📖 阅读模式，方便浏览字幕内容
- ⏱️ 时间戳跳转，点击字幕跳转到对应视频位置
- 💾 支持下载字幕

## 📦 安装

1. 下载本仓库代码或 `git clone`
2. 打开 Chrome，进入 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本仓库文件夹

## 🛠️ 技术栈

- Manifest V3
- Chrome Side Panel API
- Chrome Scripting API
